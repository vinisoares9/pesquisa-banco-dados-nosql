# Banco de Dados NoSQL: Flexibilidade e Desempenho

📚 Trabalho acadêmico desenvolvido na disciplina **Banco de Dados II**
🎓 Curso: Ciência de Dados e Machine Learning
🏫 Centro Universitário de Brasília (UNICEUB)

## 📌 Sobre o trabalho

Este artigo apresenta uma análise teórica e comparativa sobre bancos de dados NoSQL, abordando:

* Conceitos fundamentais de NoSQL
* Categorias (Chave-Valor, Documentos, Colunares e Grafos)
* Relação com OLTP e OLAP
* Comparação técnica entre MongoDB e PostgreSQL
* Cenários de uso e arquitetura híbrida (Polyglot Persistence)

## 🔎 Principais tópicos abordados

* Teorema CAP
* Consistência eventual vs consistência forte
* Escalabilidade horizontal
* Transações ACID
* Performance e flexibilidade de esquema
